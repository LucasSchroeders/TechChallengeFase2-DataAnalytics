# Previsão da Direção do IBOVESPA

Projeto desenvolvido para um desafio acadêmico da pós-graduação em **Data Analytics (FIAP)**, com o objetivo de prever se o **IBOVESPA** fechará em **alta** ou **baixa** no pregão seguinte com base em dados históricos diários.

## Sobre o desafio

O desafio consiste em construir um modelo preditivo a partir de pelo menos 2 anos de dados do índice, passando por etapas de preparação, modelagem e avaliação dos resultados.

## O que foi feito

Para atender ao desafio, foi desenvolvido um pipeline com as seguintes etapas:

- coleta e organização dos dados históricos do IBOVESPA;
- tratamento e preparação da base;
- criação da variável alvo com base na direção do fechamento do dia seguinte;
- treinamento e comparação entre modelos de classificação;
- avaliação de desempenho com métricas apropriadas para o problema.

## Resultado

O projeto entrega um processo estruturado e reproduzível para análise preditiva do IBOVESPA, com comparação entre modelos e justificativa técnica para a escolha da abordagem final.
